# VueGit
## Backend setup
```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
## Frontend install
```
cd frontend
npm install
npm run serve
```
## MongoDB tables
```
Для нормальной авторизации следует добавить табличку с пользователями к себе в БД. 
Таблица с данными прикреплена в папке DBTables: testDB_Velocimetry_v2.users.json
Также рекомендуется использовать для запуска приложения, прикрепленные таблицы БД, в которых сделан единый формат ID, 
использующихся для связей. В измененных табличках присутствует также пару тестовых записей, не относящихся к первоначальной версии БД, 
ключевое изменение только в отсутствии смешанного формата у определенных ID.
Помимо этого рекомендуется добавить таблицу для работы логов
Она также прикреплена в папке DBTables: testDB_Velocimetry_v2.activity_logs.json
```
